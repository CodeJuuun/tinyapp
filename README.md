## Dependencies used
- Node.js
- Express
- EJS
- Morgan
- Cookie-session
- bcryptjs
- Mocha & Chai

## Usage

**Install it:**
- Install all required dependencies via `npm install (dependency name)`
- Run the development web server using the `npm start` command.
- Run `npm test` to begin testing
